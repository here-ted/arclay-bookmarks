# Arclike Bookmarks

[English](README.md)

Arclike Bookmarks 是一款基于 Chrome Manifest V3 的侧边栏浏览器扩展，旨在为你提供类似 Arc 浏览器的标签页和书签管理体验。它将书签和当前打开的标签页巧妙地整合到一个垂直侧边栏中，让你可以更轻松地浏览、整理和组织你的网页。

## 🌟 主要特性 (Features)

- **侧边栏无缝集成 (SidePanel)**: 与浏览器完美融合，在侧边随时待命且不遮挡网页内容。
- **直观的书签树管理**: 以树状结构展示所有书签和文件夹，支持无缝展开和折叠。
- **动态标签页列表**: 独立展示当前窗口开启（且未保存）的标签页，轻松把控当前工作流。
- **支持拖拽交互 (Drag & Drop)**: 
  - 自由拖拽排序书签、将书签移入移出文件夹。
  - 将未保存的标签页直接拖拽进书签树区域，实现一键分类与收藏。
- **布局高度自定义**: 书签和标签页区域之间配有可拖拽调整的极简分割线，可根据你的工作偏好自定义上下区块的高度比例。
- **保持实时同步**: 书签和标签页的状态改变会及时与原生的浏览器系统保持双向数据同步更新。

## 🚀 安装说明 (Installation)

由于本扩展尚未发布到 Chrome Web Store，请通过“开发者模式”进行本地安装测试：

1. 克隆或下载本仓库源代码到本地计算机。
2. 打开基于 Chromium 的浏览器（例如 Chrome 或者 Edge）。
3. 在地址栏输入 `chrome://extensions/` （或对应的浏览器扩展管理地址）并访问。
4. 确保在页面右上角开启 **开发者模式 (Developer mode)**。
5. 点击 **加载已解压的扩展程序 (Load unpacked)** 按钮。
6. 选择本仓库所在的主目录文件夹。
7. 安装完成后，在浏览器工具栏中点击本扩展的图标，或将其固定以便随时打开 Arclike Bookmarks 侧边栏！

## 🛠️ 技术原理 (Tech Stack)

* **纯前端实现**: 整体架构基于 Vanilla JavaScript、原生 HTML 和 CSS 构建，无需 Webpack 或繁杂的依赖环境，轻量且快速。
* **Manifest V3 标准**: 紧跟最新扩展开发规范获取更优的安全与性能优势。
* **核心 Chrome Extension APIs**:
  * `chrome.sidePanel`: 处理固定侧边栏视图。
  * `chrome.bookmarks`: 读写及组织原生书签数据。
  * `chrome.tabs`: 管理浏览器的标签页活动。
  * `chrome.storage.session`: 高效处理会话层级的数据映射缓存。
